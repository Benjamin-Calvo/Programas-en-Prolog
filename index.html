Programas en Prolog

Esta es una colección de programas simples escritos en Prolog para ser usados en el curso TI-3404 Lenguajes de Programación del Instituto Tecnológico de Costa Rica. Para clonar el repositorio, se hacen los siguientes pasos:

$ git clone git@github.com:andreifu/Programas-en-Prolog.git

El código presentado debe ser usado con fines puramente didácticos.
